# Maracha-Data-Science-Project
This is the Repo for Muni Labs,  developing a system for Electronic Health Records to generate insights from the handwritten data sets. it aims at providing a way of digitizing the manual records at the Hospital.

Extract the hidden files and keep them in your root project folder; they help to provide other instances.
Run the project by opening the run.py on a command prompt.
